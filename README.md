# 🐄 Livestock Monitoring System

A real-time smart livestock monitoring system using computer vision and IoT principles.

## 🚀 Features

- 📷 Camera-based movement detection (OpenCV)
- 🧠 Behavior-based health monitoring
- ⚠️ Smart alert system (Telegram integration)
- 📊 Live dashboard (Streamlit)
- ☁️ Cloud deployment (Render)
- 🔄 Reliability monitoring (device failure detection)

## 🏗️ Architecture

Camera → Movement Detection → Backend API → Alert Engine → Dashboard

## 🧪 Use Case

Detect early signs of illness based on:
- Low activity
- Abnormal behavior patterns
- Combined health signals

## ⚙️ Tech Stack

- Python
- OpenCV
- Flask
- Streamlit
- SQLite
- Render (cloud)
- Telegram API

## 📈 Future Improvements

- Multi-animal tracking
- Temperature sensor integration
- AI-based behavior classification
- Edge deployment (Raspberry Pi)
